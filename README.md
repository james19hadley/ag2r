# AG2R — Antigravity 2.0 Remote

A lightweight mobile remote interface for monitoring and interacting with [Antigravity](https://antigravity.dev) AI coding sessions.

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start the server (requires Antigravity running with CDP enabled)
node server.js

# Antigravity must be started with:
# antigravity . --remote-debugging-port=9000
```

Then open `https://localhost:3000` on your phone (same network).
